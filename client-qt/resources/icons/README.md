## Технические требования

### Размеры
- Рекомендуемый размер: 16x16, 24x24, 32x32, 48x48, 64x64 пикселей
- Максимальный размер: 256x256 пикселей (для иконки приложения)
- Минимальный размер: 16x16 пикселей

### Формат
- Формат: PNG с прозрачным фоном
- Глубина цвета: 24-бит + альфа-канал
- Сжатие: Без потерь

### Стиль
- Стиль: Минималистичный, плоский дизайн
- Линии: Четкие, толщиной 2-3 пикселя для иконок 24x24
- Цвета: Яркие, насыщенные, соответствующие теме приложения
- Пропорции: Упрощенные, иконографические

## Рекомендации по созданию

1. Используйте векторные графические редакторы (Inkscape, Illustrator) для создания
2. Сохраняйте исходники в векторном формате (SVG)
3. Экспортируйте в PNG с высоким качеством
4. Проверяйте читаемость на маленьких размерах
5. Следите за единообразием стиля во всем наборе
6. Используйте стандартные символы и знаки для лучшего распознавания
7. Делайте акцент на простоте и ясности
