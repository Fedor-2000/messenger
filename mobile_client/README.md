# Мобильный клиент мессенджера

Это мобильное приложение для мессенджера, созданное с использованием Flutter.

## Особенности

- Поддержка iOS и Android
- Подключение к серверу через WebSocket
- Аутентификация пользователей
- Отправка и получение сообщений в реальном времени
- Хранение настроек пользователя

## Требования

- Flutter SDK
- Dart SDK

## Установка

1. Убедитесь, что Flutter установлен:
```bash
flutter doctor
```

2. Установите зависимости:
```bash
flutter pub get
```

3. Запустите приложение:
```bash
flutter run
```

## Архитектура

- `lib/main.dart` - основной файл приложения
- `lib/screens/` - экраны приложения
- `lib/services/` - сервисы для работы с API
- `lib/models/` - модели данных
- `lib/widgets/` - переиспользуемые виджеты

## Конфигурация

Для подключения к серверу измените URL в файле `lib/main.dart`:

```dart
Uri.parse('ws://your-server-address:8080/ws')
```

## Сборка

Для создания релизной версии:

```bash
# Android
flutter build apk --release

# iOS
flutter build ios --release
```
